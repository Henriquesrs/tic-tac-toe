# Tic-Tac-Toe (Jogo da Velha) — Vite + React

Descrição: Implementação do jogo da velha usando React e Vite, com histórico de jogadas, reinício e indicação de vencedor.

Tecnologias:
- React
- Vite
- CSS

Como rodar:
1. `npm install`
2. `npm run dev`
3. Abrir `http://localhost:5173`

Principais aprendizados:
- Componentização com React
- Estado com useState
- Manipulação de eventos e props
